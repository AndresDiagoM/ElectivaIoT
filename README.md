# ElectivaIoT

# PRÁCTICAS IoT

### Objetivo
Estas prácticas tienen como objetivo familiarizarse con el uso de tecnologías de internet para desplegar servicios en la nube para emplearlos con aplicaciones IoT.

## Ramas del repositorio

 * Rama Main: En esta rama se encuentra la documentanción en un archivo .md, de cada una de las prácticas, junto con evidencia de pantallazos.
 * Rama Hexagonal: En esta rama se encuentra la aplicación de la arquitectura Hexagonal al proyecto, junto con la carpeta del proyecto desarrollada en NestJS.
 * Rama HerokuD: En esta rama estan los archivos del protecto de NestJS, para facilitar el deploy en heroku.
